# ✊✋✌️ Rock Paper Scissors Game (Python)

A fun and interactive GUI-based Rock Paper Scissors game built using Python and Tkinter with image-based gameplay.

---

## Features

✔ Play against the computer

✔ Random computer moves

✔ Live scoreboard tracking

✔ Image-based buttons (Rock, Paper, Scissors)

✔ Instant result display (Win/Lose/Tie)

✔ Simple and clean GUI

---

## 🛠️ Tech Stack

* Python
* Tkinter
* Pillow (for images)
